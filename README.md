# コーヒー＆スイーツ ペアリング・ログ ☕🍰

あなたのコーヒータイムをより豊かに。その日の気分や飲み物に合わせて、最高のスイーツを提案・記録するアプリです。

## ✨ 主な機能

### 1. パーソナライズされたペアリング提案
ブラックやカフェラテなど、6種類の飲み物から選択すると、AIやロジックに基づいたおすすめスイーツを提案します。
* **「軽め」「重め」の2パターン**: 各4種類ずつ、合計8つの候補を表示。
* **納得の理由付き**: なぜその組み合わせが合うのか、納得感のある解説を表示します。

### 2. こだわりの記録・ログ機能
提案された組み合わせを試した後は、その感動をそのまま記録できます。
* **評価システム**: 星5段階の評価 ＋ 自由な感想記入。
* **柔軟な入力**: リストにないスイーツを楽しんだ場合も、自由入力で記録可能です。
* **思い出を視覚化**: スイーツの**画像保存機能**も搭載しています。

### 3. あなたの好みを可視化する「傾向分析」
蓄積されたデータから、あなたの好みを自動で集計します。
* **最多ペアリング**: どの組み合わせを一番多く楽しんでいるか。
* **高評価コーヒー**: 最も満足度が高いコーヒーの種類はどれか。

### 4. ログ管理
「全ログ表示」ページから、これまでのすべての記録を一覧で確認できます。不要なデータの削除も簡単です。

## 💡 開発の裏話（AIへの指示と苦労した点）

### AIへの指示の出し方
「機能の羅列」を渡すだけでなく、**「ユーザーがどんな体験をするか」**という流れを具体的に伝えるようにしました。特に「リスト外のスイーツも入力できる」「画像も保存できる」といった細かいこだわりを強調して指示に含めました。

### 苦労したポイント
* **ロジックの言語化**: 「軽め・重め」の定義や、組み合わせの理由を違和感なく説明させるための文言調整に苦労しました。
* **分析機能の表現**: データの自動集計という少し複雑な機能を、README上でいかに「便利そう」に見せるかという見せ方の工夫に時間をかけました。
